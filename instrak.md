# Scriptsoft Project Setup Instructions

මෙම ගොනුව මගින් Scriptsoft වෙබ් අඩවිය සාදාගන්නා ආකාරය පියවරෙන් පියවර විස්තර කරයි.

## 📂 ෆයිල් එක සකසා ගන්නා ආකාරය
1. ඔබේ පරිගණකයේ අලුත් Folder එකක් සාදන්න (නම: `scriptsoft_web`).
2. එම Folder එක ඇතුළත `index.html` නමින් අලුත් ෆයිල් එකක් සාදන්න.
3. ඉහත ලබා දී ඇති HTML කෝඩ් එක සම්පූර්ණයෙන්ම කොපි කර `index.html` තුළ Paste කර Save කරන්න.

## 🛠 භාවිතා කර ඇති තාක්ෂණයන් (Technologies)
- **HTML5**: වෙබ් අඩවියේ ව්‍යුහය සඳහා.
- **Bootstrap 5**: Responsive design සහ layout එක පහසුවෙන් සාදා ගැනීමට.
- **Custom CSS**: වර්ණ සහ විශේෂ හැඩතල (Hero gradient, Card effects) සඳහා.
- **AOS (Animate On Scroll)**: වෙබ් අඩවිය පහළට scroll කරන විට එන animations සඳහා.

## 🚀 Animations වෙනස් කරන්නේ කොහොමද?
වෙබ් අඩවියේ ඕනෑම තැනකට animation එකක් එක් කිරීමට අදාළ HTML tag එක ඇතුළත `data-aos` attribute එක භාවිතා කරන්න.
* `data-aos="fade-up"` - ඉහළට මතු වීම.
* `data-aos="zoom-in"` - විශාල වී පෙනීම.
* `data-aos="flip-left"` - වමට කැරකීම.

## 🌐 වෙබ් අඩවිය පරීක්ෂා කිරීම
`index.html` ෆයිල් එක ඕනෑම Web Browser එකකින් (Chrome, Brave, Firefox) open කර බලන්න.

## 📧 වැඩිදුර වෙනස්කම්
ඔබේ සිස්ටම් වල පින්තූර "Portfolio" කොටසට ඇතුළත් කර වෙබ් අඩවිය තවත් අලංකාර කර ගත හැක.
